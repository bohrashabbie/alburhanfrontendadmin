# AL-Burhan Admin Panel

React + TypeScript admin panel for managing all AL-Burhan Regional website content.

## Setup

```bash
npm install
npm run dev
```

Runs on http://localhost:5173 with API proxy to http://localhost:8001.

## Login

- **Username:** admin
- **Password:** admin123

## Features

- Dashboard with entity counts
- Full CRUD for all content types: Site Settings, Navigation, Carousel, Page Contents, Services, Sectors, Team Members, Countries, Branches, Contact Info, Brands, Products, Banners, Project Categories, Projects, Social Links, Footer Links, Static Pages
- Contact form submissions viewer with read/unread tracking
- JWT authentication with protected routes
- Bilingual content editing (EN/AR)
